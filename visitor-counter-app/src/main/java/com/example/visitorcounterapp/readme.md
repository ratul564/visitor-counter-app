# URL hit counter <br>
This is a Spring Boot project. with the help of this project we can easily find out traffic by user, How  many time user visit in web given browser.
# Dependency Used
* Spring Web
* Spring Dev Tools
# Annotation used
* @Service
* @RestController
* @RequestMapping
* @Getmapping
* @PostMapping
* @PutMapping
* @DeleteMapping
* @Autowired
